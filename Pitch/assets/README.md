## Format-Empfehlungen

- **Pixelformat:** PNG für UI-Screenshots, JPEG für Fotos
- **Auflösung:** mindestens 1920×1080 für 16:9-Folien (besser 2560×1440 für Retina-Beamer)
- **Branding:** Lesekumpel-Coral `#D67171` und Gelb `#FFD54F` als Akzent verwenden
