Kinder, die mit zwei Sprachen aufwachsen, können von einem erhöhten metalinguistischen Bewusstsein profitieren. Sie lernen früh, dass Dinge in verschiedenen Sprachen unterschiedlich benannt werden. Dieses Bewusstsein für die Willkürlichkeit sprachlicher Zeichen kann sich positiv auf die phonologische Bewusstheit auswirken — eine Schlüsselkompetenz für das Lesenlernen.
