Typografie

Fredoka für Überschriften — rund, freundlich. Quicksand für Fließtext — klar, lesbar.

Der schnelle braune Fuchs springt über den faulen Hund. Lesen macht Spaß!

Hover & Transitions

Sanfte Bewegungen mit Charakter — Buttons haben einen leichten Bounce-Overshoot.

Element Hover-Effekt
Buttons translateY(-3px) scale(1.02) + verstärkter Schatten
Karten translateY(-4px) + shadow-card-hover
Links Farbwechsel zu --accent-coral 
/* Karten — sanft */
transition: transform 0.3s ease, box-shadow 0.3s ease;

/* Buttons — Bounce-Overshoot */
transition: all 0.3s cubic-bezier(0.34, 1.56, 0.64, 1);

💡 Fahre mit der Maus über die Komponenten oben, um die Effekte live zu sehen.

Responsive & Barrierefreiheit

Mobile-First, Touch-tauglich, für alle Lesefähigkeiten zugänglich.

📱

Responsive

  • Mobile-First Ansatz
  • Breakpoint: 768px min-width
  • Desktop: Column → Row, mehr Padding
  • Touch-Targets: mindestens 44px Höhe
♿

Accessibility

  • Legasthenie-Modus (Lexend als Alternative)
  • Hoher Kontrast-Modus
  • Schriftgrößen-Anpassung
  • Coral nie für kleinen Fließtext
  • Fokus-Ringe: 0 0 0 3px rgba(214,113,113,0.4)
